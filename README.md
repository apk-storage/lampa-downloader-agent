# Lampa Downloader

**Скачивайте торренты с ТВ прямо на свой компьютер — одной кнопкой из Лампы.**

Выбираете фильм в [Лампе](https://lampa.mx) → жмёте «Скачать на ПК» → файл качается на ваш компьютер.
Без облаков и посредников: всё передаётся через сквозное шифрование.

</div>

---

## Как это работает

```
  ТВ / телефон (Лампа)              релей (только шифротекст)             ваш ПК (агент)
 ┌───────────────────┐   E2E    ┌───────────────────────┐   E2E    ┌───────────────────┐
 │  плагин: кнопка   │ ───────► │   relay.bwa.ad        │ ───────► │  качает торрент   │
 │  «Скачать на ПК»  │          │  (видит только шифр)  │          │  в вашу папку     │
 └───────────────────┘          └───────────────────────┘          └───────────────────┘
```

- **Плагин** в Лампе шифрует магнит-ссылку и отправляет её.
- **Релей** — простая «труба»: пересылает шифротекст и не может его прочитать.
- **Агент** на вашем ПК расшифровывает и качает файл встроенным торрент-движком.

Шифрование сквозное (NaCl box, X25519). Ключи генерируются на ваших устройствах, релей их не видит.

## Установка

### 1. Поставьте плагин в Лампу

Настройки → Расширения → Добавить плагин:

```
https://apk-storage.github.io/lampa-plugins/plugin.js
```

### 2. Скачайте и запустите агент на ПК

| ОС | Файл |
|----|------|
| **Windows** | [lampa-agent.exe](https://github.com/apk-storage/lampa-downloader-agent/releases/latest/download/lampa-agent.exe) |
| **macOS** (Intel) | [lampa-agent-macos-intel](https://github.com/apk-storage/lampa-downloader-agent/releases/latest/download/lampa-agent-macos-intel) |
| **macOS** (Apple Silicon) | [lampa-agent-macos-arm](https://github.com/apk-storage/lampa-downloader-agent/releases/latest/download/lampa-agent-macos-arm) |
| **Linux** | [lampa-agent-linux](https://github.com/apk-storage/lampa-downloader-agent/releases/latest/download/lampa-agent-linux) |

Запустите — откроется окно с 6-значным кодом.

> **Windows покажет «Windows защитила ваш компьютер».** Это нормально для бесплатных программ без платной подписи.
> Нажмите **«Подробнее» → «Выполнить в любом случае»**.

### 3. Свяжите ПК с Лампой

В Лампе: Настройки → **Lampa Downloader** → **Подключить ПК** → введите 6-значный код с экрана агента.

### 4. Качайте

Откройте любой фильм → Торренты → удерживайте раздачу → **«Скачать на ПК»**.
Прогресс виден и в окне агента, и в Лампе (Настройки → Lampa Downloader → Загрузки).

## Возможности

- 🔒 Сквозное шифрование — релей видит только шифротекст
- 🖥️ Нативное приложение с иконкой в трее, автозапуском и фоновой работой
- 📊 Прогресс закачек на ТВ и в окне агента
- 📁 Выбор папки загрузок, раскладка по категориям (Фильмы / Сериалы)
- ⏸️ Докачка после перезапуска, отмена закачек
- 🔁 Работает за NAT (агент сам подключается к релею)

## Сборка из исходников

Нужен [Go 1.24+](https://go.dev/dl/). Сборка не требует C-компилятора (чистый Go).

```bash
git clone https://github.com/apk-storage/lampa-downloader-agent
cd lampa-downloader-agent
go get github.com/anacrolix/torrent@v1.60.0
go build -ldflags "-H=windowsgui" -o lampa-agent.exe .   # Windows, фоновый
```

Сборки под все ОС собираются автоматически в GitHub Actions при каждом коммите.

## Безопасность и приватность

- Магнит-ссылки шифруются на устройстве и расшифровываются только вашим агентом.
- Релей не хранит и не может прочитать содержимое.
- Код агента открыт — можете собрать сами.
